核心功能

精准、纯净、高效

专为 AI 工作流设计的内容提取工具

精准提取

支持「框选区域」和「整页提取」两种模式，智能识别正文、代码块和表格，自动过滤广告和导航干扰。

多格式输出

完美支持 Markdown、JSON、XML 格式，直接适配 ChatGPT、Claude 等大语言模型的输入需求。

图文打包

支持导出 ZIP 压缩包，包含 Markdown 文件和本地化图片，完整保留上下文信息。

快捷键支持

使用 Cmd+Shift+X 快速激活元素选择模式，移动鼠标即可高亮并提取。

使用指南

三步开始使用

离线安装，无需商店审核

01

下载源码

从 GitHub 下载或克隆本项目代码到本地

git clone https://github.com/let5sne/SnowClip.git
02

加载扩展

打开 Chrome 访问 chrome://extensions/，开启「开发者模式」，点击「加载已解压的扩展程序」

03

选择目录

选择项目中的 src/browser-extension 文件夹，即可完成安装

使用方式

框选提取 点击扩展图标 → 选择格式 → 点击「框选区域提取」→ 拖拽选择
整页提取 点击「提取整页内容」自动识别并提取正文
快捷键模式 按 Cmd+Shift+X 激活，悬停高亮，点击提取
